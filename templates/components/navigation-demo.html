Navigation Components Demo

This page demonstrates the three navigation components: Skip Links, Main Navigation, and Breadcrumbs. All components follow Web Standards and are fully accessible.

1. Skip Links Component

Skip links allow keyboard users to jump directly to main content areas. Press Tab to see them appear at the top of the page.

Features:
  • ‚úÖ Hidden until focused (keyboard navigation)
  • ‚úÖ Jump to main content, navigation, or breadcrumbs
  • ‚úÖ High contrast for visibility
  • ‚úÖ Proper focus indicators

2. Main Navigation Component

Context-aware navigation menu with progressive disclosure. Only shows items that are available in the current context.

Features:
  • ‚úÖ Current page indication with aria-current="page"
  • ‚úÖ Icon support for visual clarity
  • ‚úÖ Badge support for notifications
  • ‚úÖ Nested submenu support
  • ‚úÖ Minimum 44x44px touch targets
  • ‚úÖ Keyboard accessible (Tab, Enter, Space)
  • ‚úÖ Mobile responsive (stacks vertically)

3. Breadcrumbs Component

Shows the user's location in the information hierarchy. Helps users understand where they are and navigate back.

Features:
  • ‚úÖ Current page marked with aria-current="page"
  • ‚úÖ Separators are aria-hidden (not announced to screen readers)
  • ‚úÖ Proper semantic HTML with <nav> and <ol>
  • ‚úÖ Keyboard accessible links
  • ‚úÖ Responsive (smaller text on mobile)

Accessibility Testing

Test these components with keyboard and screen readers:

Keyboard Testing:
  1. Press Tab to see skip links appear
  2. Press Enter on a skip link to jump to that section
  3. Press Tab through navigation items
  4. Press Enter or Space to activate links
  5. Hover over "Services" to see submenu (or use keyboard)

Screen Reader Testing:
  • Navigation is announced as "Main navigation"
  • Current page is announced as "current page"
  • Breadcrumbs are announced as "Breadcrumb navigation"
  • Separators are not announced (aria-hidden)
  • Badge count is announced descriptively

Usage in Templates

How to use these components in your Jinja2 templates: 

{# Skip Links #}
{% include 'components/skip-links.html' %}

{# Breadcrumbs #}
{% set breadcrumbs = [
    {'label': 'Home', 'url': '/', 'is_current': False},
    {'label': 'Services', 'url': '/services', 'is_current': False},
    {'label': 'Apache', 'url': '/services/apache', 'is_current': True}
] %}
{% include 'components/breadcrumbs.html' %}

{# Main Navigation #}
{% set nav_items = [
    {'label': 'Home', 'url': '/', 'is_current': True, 'is_available': True, 'icon': 'home'},
    {'label': 'Workstation', 'url': '/workstation', 'is_current': False, 'is_available': True, 'icon': 'server'},
    {'label': 'Services', 'url': '/services', 'is_current': False, 'is_available': True, 'icon': 'grid', 'badge': '3'}
] %}
{% include 'components/navigation.html' %}
